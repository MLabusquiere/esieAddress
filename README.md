Usage
=====

* Build it <code>mvn clean package -Dspring.profiles.active=dev</code>
* Add <code>export JAVA_OPTS=$JAVA_OPTS" -Dspring.profiles.active=dev"</code> to $CATALINA_HOME/bin/setenv.sh
* Deployed it

* Tested on Chrome 30.0.1599.101
